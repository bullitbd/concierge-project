Concierge Add Media Help

1. General Use

The add media dialog allows "adding" external content to the Concierge Media Catalog, by referencing it's URL or Embed information. It automatically produces and saves thumbnails for stored image [web pages, documents, html] URLs, and provides for the entry of an image URL to be processed as a thumbnail for any stored Video code. A variety of identifying information is collected and, when the content is "built", a link is assembled which opens the associated media in a javascript pop-up window (Floatbox, 'FB') when clicked.

Action Buttons

Please see the tooltips which appear when hovering your mouse over the buttons.

2. Embed Code  required

This field takes media information, typically copied from a video site's "embed" code, image or other content url. This information, using the Media Type information is parsed in the background for appropriate information and used in the final link building process

3. Media Type  required

In order for the parse engine to function properly, this data must be correctly identified by the following types:

  • Video: for VIDEO embed codes copied from video or partner sites.
  • R21VIDEO: used for inclusion of site content references in the catalog.
  • Image: used with standard IMAGE urls; these begin with http... and end with common image file extensions (jpg, png, etc.).

4. Submitter  required

Simply, who is entering the content. Please identify yourself so that someone else may ask you about your submission.

5. Owner / Campaign  required

This is a two-part control. Owner is an organization name - either R21 or one of its sponsors, partners, etc, as indicated in the Owner Type field. Campaign is the particular promotion of which the content entry is a part. The Owner Type field is informational and not available for input. More to come here...

6. Modify Size To:

This field defaults to 100%, meaning no change in content dimensions from the original. This value may be changed as appropriate, and each subsequent "build" request will allow a preview at the new size, as a % of the original display. This can be changed and re-build as often as necessary to arrive at the desired presentation size.

Keep in mind that this value will typically fall within the maximum range of approximately 40 to 400 (percent). Original resolution and desired effect will dictate an acceptable value. More to come here...

7. Title  required

The title appears in the center of the top border around the content. Unless specific circumstances dictate, a title is manditory, as it is the only required visual descriptor of the content, and is the default for both the accessible "alt" content and the tooltip for the link. If absolutely necessary, this field will accept a blank entry (at least one "space" character).

8. 2nd Caption

The 2nd Caption appears in the center of the lower border. It's use is typically to act as a category title for a group of images, or to impart more information to a single piece of content.

10. Description  required

The description field is mandatory, and serves as one of the primary searchable fields in the catalog. Typically, this might be copied and pasted from the site where the media lives, however, any intentionally helpful descriptive text can be entered here.
